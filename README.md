# Nikon Z6III Rehberi

Nikon Z6III için kişisel kullanma kılavuzu ve çekim notları uygulaması. Kullanım Kılavuzu ile Başvuru Kaynağı'nı tek bir aranabilir arayüzde birleştirir; ayrıca video/fotoğraf ayarları, flaş, gimbal, mikrofon ve kurgu üzerine kişisel notları içerir.

Telefonda kullanmak için: siteyi Safari/Chrome'da açıp "Ana Ekrana Ekle" ile PWA olarak yükleyebilirsiniz. Çevrimdışı çalışır.

## Geliştirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

`main` dalına yapılan her push, GitHub Actions ile otomatik olarak GitHub Pages'e deploy edilir.
